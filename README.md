## SpringBoot - vue.js 연동 및 gradle 빌드 및 실행
### 환경
- Framework : Spring Boot
- Bulid : Gradle
- JDK : JDK 8
- Front : Vuejs

- Spring Boot + Vuejs를 하나의 프로젝트로 관리하고자 할때 One Build가 가능하도록 프로젝트를 구성하는 것
- Bulid 진행시 Proxy를 통한 자동화가 가능하도록  프로젝트를 구성하는 것

### 설명
1. vue.js 설정
